## ⚡ About Me

I am a Full-Stack Developer intern passionate about creating modern and efficient web applications. My daily focus is on developing robust solutions using **NestJS** and **Angular**, always striving to learn and apply **good architecture** and **best coding practices**.

I love exploring different technologies and currently work with a diverse stack, including relational and NoSQL databases like **MySQL** and **MongoDB**, as well as containerizing applications with **Docker**. I'm constantly challenging myself to build better, more scalable software and take the next step in my career.

**When I'm not coding:** You can usually find me listening to music on Spotify or playing some competitive game to unwind.
